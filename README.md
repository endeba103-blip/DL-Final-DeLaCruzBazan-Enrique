# Proyecto Final - Deep Learning

## 📌 Descripción
Este proyecto tiene como objetivo clasificar imágenes de ropa utilizando una Red Neuronal Convolucional (CNN) aplicada al dataset Fashion MNIST.

## 🧠 Modelo
Se implementó una CNN con:
- Capas Conv2D
- MaxPooling
- Capas densas

## 📊 Resultados
El modelo alcanzó una precisión aproximada del 90% en el conjunto de prueba.

## ▶️ Ejecución
1. Abrir el notebook en Google Colab
2. Ejecutar todas las celdas
3. Visualizar resultados

## 📂 Dataset
Fashion MNIST (incluido en TensorFlow)
